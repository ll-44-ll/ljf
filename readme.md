一、代码制作流程文字简介
这段天气预报页面分三层制作，步骤简单清晰：
搭建 HTML 页面框架：写出页面标题、定位按钮、城市输入搜索框、错误提示文字区域、天气展示板块，划分实时天气和三日预报两块内容，给每个交互、展示标签设置唯一标识。
编写 CSS 美化页面：统一设置全局配色，用弹性布局让卡片居中适配手机，添加渐变背景、半透明毛玻璃卡片，给按钮、输入框做点击悬浮动画，天气模块默认隐藏，查询后淡入弹出。
编写 JavaScript 交互逻辑：
① 建立天气数字代码和中文文字的对应表；
② 封装接口请求函数，先把城市名称转换成经纬度，再根据坐标获取实时温度和三日预报；
③ 绑定点击、回车、页面加载等交互事件，实现手动搜城市、一键定位本地天气；
④ 加入本地存储功能，保存上次查询的城市，页面刷新不用重复输入；
⑤ 增加异常捕获，网络错误、找不到城市、定位被拒绝时给出中文提示。
二、各部分运用属性中文解说
（一）HTML 标签使用的属性
lang 属性：写在 html 标签上，声明页面文字为中文；
charset 属性：设置文字编码，防止中文出现乱码；
viewport 属性：适配手机，让页面宽度贴合手机屏幕，禁止页面自动放大缩小；
class 属性：给标签绑定样式，批量控制布局、颜色；
id 属性：给每个元素独一无二的标记，方便 JS 找到并修改内容；
type 属性：输入框专用，限定只能输入普通文字；
placeholder 属性：输入框空白时，显示浅色提示文字引导用户输入城市。
（二）CSS 样式运用的属性
自定义变量属性：统一存放主题色、背景渐变，全页面可以重复调用，改颜色只需改一处；
弹性布局相关属性：display 弹性盒子、水平居中、垂直居中、两端分散对齐、元素间距、输入框自动占满剩余宽度；
页面基础属性：最小高度铺满浏览器窗口、字体、内外边距、背景渐变；
卡片美化属性：半透明背景、圆角、阴影、毛玻璃模糊效果、限制卡片最大宽度；
交互过渡属性：过渡动画，让按钮变色、输入框变色过程平滑；
伪类属性：输入框点击激活样式、鼠标悬浮按钮变色；
显示控制属性：默认隐藏天气、错误提示板块；
动画属性：透明度、上下位移，实现天气卡片淡入滑出动画。
（三）JavaScript 运用的属性与内置功能
元素获取功能：根据 id 找到页面上的输入框、按钮、展示板块；
输入框相关属性：读取用户输入的文字、清除文字前后多余空格；
网络请求功能：调取免费天气网络接口，获取全国城市气象数据；
异步处理语法：同步写法处理网络请求，等待数据返回再渲染页面；
页面修改属性：修改页面文字、拼接预报板块完整代码插入页面；
数据处理功能：数字温度取整数、截取日期只保留月和日；
定位功能：调用浏览器自带定位工具，获取用户当前位置的纬度、经度；
本地存储属性：保存上次查询的城市，页面重新打开自动读取；
事件监听功能：监听按钮点击、键盘回车、页面加载完成动作，触发对应查询逻辑；
异常捕获功能：拦截网络、定位、查询失败问题，弹出中文提示，页面不会卡死空白。
